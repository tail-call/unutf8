# unutf8
Convert utf-8 to stream of code points

## Synopsis 

    unutf8 < input > output
    
## Usage

    $ echo привет, мир ♥ | unutf8
    1087
    1088
    1080
    1074
    1077
    1090
    44
    32
    1084
    1080
    1088
    32
    9829
    10
    $ _

## Building

    make unutf8
